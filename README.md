# Satellite Image Dashboard

A simple Vue.js dashboard for displaying satellite images and metadata.

## Setup

1.  **Frontend:**

    ```bash
    cd frontend
    npm install
    npm run serve
    ```

2.  **Backend:**

    ```bash
    cd backend
    npm install
    npm start
    ```

## Running Tests

**Frontend:**
```bash
cd frontend
npm run test:unit
```

**Backend:**
```bash
cd backend
npm run test
```
